# recipe-develop — 레시피 개발(초안)

## 할 일
후보 하나를 받아 `explore/candidates/<이름>/draft.md`로 저장한다.

## draft.md 구성 (구체적 수치 필수)
- **개요**: 콘셉트·타깃·차별점·발효 유형(진짜김치/가공품)
- **재료** (4인분 기준): 항목 + 분량(g·통·큰술)
- **절임**: 소금 농도(%)·시간·온도
- **만드는 법**: 5~7단계
- **발효 일정**: 실온/냉장 온도·기간·먹기 좋은 시점
- **플레이팅**: 색·담음새·촬영각
- **실패 포인트**: 물러짐·과발효 등 주의 + 회피법

## 원칙
- 분량·시간·온도는 **반드시 수치**. 모르면 추정이라 밝히고 실습으로 검증할 값으로 표시.
- 외부 레시피 참고 시 자기 말로 재구성 + 출처. 통째 복사 금지.
- 초안임을 명시 — 실습(`recipe-trial`)으로 보정한다.
