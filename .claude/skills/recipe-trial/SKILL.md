# recipe-trial — 실습(담그기) 기록

## 할 일
`explore/candidates/<이름>/trials/<MMDD>.md`로 저장한다.

## 기록 항목
- **담근 날짜·분량**
- **draft 대비 변경**: 무엇을 얼마나 바꿨나 (예: 소금 1큰술↓, 배 추가)
- **관찰**: 색·냄새·식감·국물 상태
- **사진 메모**: 파일명·각도(있으면)
- **다음에 바꿀 점**: 구체적으로

## 후속
- 변경이 유의미하면 `draft.md`의 수치를 갱신(이력은 trial에 남김).
- 발효 며칠 뒤 → `recipe-taste`로 시식 평가.
