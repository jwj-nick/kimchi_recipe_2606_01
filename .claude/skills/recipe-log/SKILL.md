# recipe-log — 세션 로그 정리

## 규칙
- 파일: `chatlog/MMDD-Chat.md` (예: `0613-Chat.md`). 없으면 생성, 있으면 **이어서 누적**.
- 형식(라운드별):
  - `### 사용자 원문` — 사용자가 채팅창에 적은 **원문 그대로** (코드펜스로 감쌈)
  - `### Claude 산출물 (파일/폴더)` — 그 라운드에 만든/바꾼 파일·폴더 간략 기록
  - `### 다음 단계` — 다음에 할 일

## 할 일
1. 이번 세션의 라운드들을 위 형식으로 작성/추가한다.
2. 사용자 원문은 요약하지 말고 그대로 옮긴다.
3. 마지막에 "다음 단계"를 명확히 — 다음 세션 `recipe-start`가 이걸 읽는다.
