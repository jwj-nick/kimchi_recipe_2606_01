# recipe-taste — 시식 평가

## 할 일
`explore/candidates/<이름>/tasting/<MMDD>.md`로 저장한다.

## 루브릭 (각 3점)
| 항목 | 기준 |
|---|---|
| 맛·향 | 균형·감칠맛·발효취 |
| 식감 | 아삭함·숙성 정도 |
| 비주얼 | 색·플레이팅·촬영각 |
| 발효 | 보존성·산도 변화 |
| 차별성 | 스토리·시장성 |

**타깃별 가중**: 외국인=맛·향 / 참신=차별성 / 건강=발효 / 시니어=식감·삼킴 / 신세대=비주얼.

## 출력
- 점수표(점수 칸 + 코멘트 칸) + 가중 합산
- **개선점** 1~3개
- **승격 판단**: 통과면 `recipe-promote` 권유, 아니면 다음 trial 방향.

## 비교
- 여러 후보 누적 시 `explore/candidates/*/tasting/`을 모아 순위표·1순위 추천.
