# recipe-start — 세션 시작·현황 점검

## 할 일
1. `CLAUDE.md`와 `data.md`(분류·루브릭)를 읽어 규칙·데이터 모델을 확인한다.
2. 최신 `chatlog/MMDD-Chat.md`의 마지막 라운드 "다음 단계"를 읽는다.
3. 진행 상태를 스캔한다:
   - `explore/candidates/*/` — 개발(draft.md)·실습(trials/)·시식(tasting/) 어느 단계인지.
   - `recipes/*/` — 승격된 확정 레시피.
4. **결론부터** 보고: 지금 상태 한 줄 + 다음에 할 한 가지(추천 스킬과 함께).

## 출력
- 후보별 단계 표(이름 / 단계 / 다음 액션)
- 추천 다음 스킬: explore·develop·trial·taste·promote 중 하나
